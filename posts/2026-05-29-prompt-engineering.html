Prompt Engineering

2026-05-29 · 10 min de lectura · prompt engineering inteligencia artificial desarrollo de software

Introducción al Prompt Engineering

El prompt engineering es una técnica que se utiliza para mejorar la eficiencia y la precisión de los modelos de lenguaje en la inteligencia artificial. Los prompts son las instrucciones o preguntas que se le dan a un modelo de lenguaje para que genere una respuesta. El objetivo del prompt engineering es diseñar prompts que sean claros, concisos y efectivos para obtener las respuestas deseadas.

Tipos de Prompts

Existen diferentes tipos de prompts que se pueden utilizar dependiendo del objetivo y el modelo de lenguaje. Algunos de los tipos de prompts más comunes son:

  • Prompts de pregunta: Son utilizados para hacer preguntas directas al modelo de lenguaje. Ejemplo: 
    ¿Cuál es la capital de Francia?
  • Prompts de descripción: Son utilizados para describir un tema o concepto. Ejemplo: 
    Describe el proceso de fotosíntesis en las plantas.
  • Prompts de generación: Son utilizados para generar texto o contenido. Ejemplo: 
    Genera un cuento corto sobre un viaje a la luna.

Ejemplos de Prompts

A continuación, se presentan algunos ejemplos de prompts que se pueden utilizar en diferentes contextos:

Un ejemplo de prompt para una tarea de clasificación de texto es: 
Clasifica el siguiente texto como positivo, negativo o neutral: "Me encantó el restaurante, la comida estaba deliciosa".

Otro ejemplo de prompt para una tarea de generación de texto es: 

Genera un correo electrónico de agradecimiento a un cliente por su compra.

Un ejemplo de prompt para una tarea de respuesta a preguntas es: 

¿Cuál es el significado de la palabra "inteligencia artificial"?

Trucos para Mejorar los Prompts

A continuación, se presentan algunos trucos que se pueden utilizar para mejorar los prompts:

  • Claridad: Asegúrate de que el prompt sea claro y conciso. Evita utilizar jerga o términos técnicos que puedan confundir al modelo de lenguaje.
  • Espectificidad: Asegúrate de que el prompt sea lo suficientemente específico para que el modelo de lenguaje pueda entender lo que se solicita.
  • Contexto: Proporciona contexto adicional al modelo de lenguaje para que pueda entender mejor el prompt. Por ejemplo, si estás solicitando una respuesta a una pregunta, proporciona información adicional sobre el tema o el contexto en el que se está haciendo la pregunta.

Conclusión

El prompt engineering es una técnica poderosa que se puede utilizar para mejorar la eficiencia y la precisión de los modelos de lenguaje en la inteligencia artificial. Al diseñar prompts claros, concisos y efectivos, se puede obtener respuestas más precisas y relevantes. Algunos ejemplos de prompts que se pueden utilizar son: 

Describe el proceso de entrenamiento de un modelo de lenguaje.
, 
Genera un resumen de un artículo sobre inteligencia artificial.
o 
Clasifica el siguiente texto como positivo, negativo o neutral: "Me encantó el producto, la calidad es excelente".
